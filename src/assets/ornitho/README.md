# Ornitho UI asset slots

Drop the generated dashboard assets in this directory using these canonical names:

- `logo.png`
- `sidebar-illustration.png`
- `audio-wave-banner.png`
- `dashboard-sparkles.png`
- `provider-openai.png`
- `provider-deepgram.png`
- `provider-google.png`

The current UI uses CSS and lucide-backed fallbacks until these bitmap assets are available.
